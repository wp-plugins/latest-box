==Description==
Simple plugin that allows to the user to show a box containing featured images of recent posts. 
Can be used like widget or short code 

== Installation ==
Upload the LatestBox plugin to your blog and activate - that's all.

The widget offrers you option how many posts to be displayed. 

If you need a shortcode usage, here is the snippet:
